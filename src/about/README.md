# sportDeets - About

Technical portfolio and architectural overview of the sportDeets platform.

## Purpose

This standalone React application serves as a comprehensive technical showcase for the sportDeets platform, designed for:
- Potential employers reviewing technical architecture
- Documentation of design decisions and technology stack
- Public transparency into platform capabilities

## Development

```bash
npm install
npm start
```

Runs on `http://localhost:3000` by default.

## Deployment

This app is deployed to `about.sportdeets.com` as a separate subdomain from the main application.

```bash
npm run build
```

## Architecture

Single-page application with smooth scrolling navigation through seven major sections:
1. Overview
2. Architecture
3. AI and Predictive Insights
4. Observability
5. Data Quality and Ingestion
6. DevOps and GitOps
7. Future Enhancements
